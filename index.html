Almost there!

Follow the 4 steps below — takes about 5 minutes total.

1Push the bot files to GitHub
Create a new GitHub repository (public or private — both work).
Upload all the bot files into it: bot.js, receipt.js, package.json, and the .github/workflows/bot.yml folder.
📁 your-repo/
├── bot.js
├── receipt.js
├── package.json
└── .github/workflows/bot.yml
2Add these Secrets to your GitHub repo
Go to your repo → Settings → Secrets and variables → Actions → click New repository secret for each row below:
BOT_TOKEN —
OWNER_IDS —
SHOP_NAME —
SHOP_ADDR —
SHOP_PHONE —
CURRENCY —
3Start the bot for the first time
Go to your repo → Actions tab → click OrderBot — Always On → click Run workflow → click the green Run workflow button.

The bot will start within 30 seconds. After that it restarts automatically every 5 hours — you don't need to do anything.
4Add your first products
Open Telegram and message your bot. Since you're an owner, you'll get the owner keyboard. Tap 📦 Products then send commands like:

/addproduct Nasi Lemak|8.00|25
/addproduct Teh Tarik|3.50|50

That's it! Your customers can now message the bot and place orders.
📦 Products you entered Ready
These are saved in data.json — upload this file to your GitHub repo too so the bot starts with your products pre-loaded. You can also add them via bot commands after starting.
